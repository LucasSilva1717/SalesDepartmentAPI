Backend (Java JDBC & Data Access Object): Implementation of a robust data persistence layer using the DAO design pattern to decouple business logic from database access. The technical focus consisted of manually building a CRUD (Create, Read, Update, Delete) infrastructure with pure JDBC, managing connections and transactions to ensure data integrity. The architecture was optimized using manual Dependency Injection via DaoFactory and custom exception handling to encapsulate infrastructure failures.

Tools used: Java 11+, JDBC, MySQL, SQL, Git/GitHub and VS Code IDE.
